# MyLeaf

MyLeaf is a collaborative LaTeX editor (Overleaf-style) built with Next.js.

## Docs
- Architecture: [ARCHITECTURE.md](ARCHITECTURE.md)
- Requirements: [REQUIREMENTS.md](REQUIREMENTS.md)

## Getting Started (Dev)
```bash
npm install
npm run dev
```

Open http://localhost:3000.

Start editing the UI at [src/app/page.tsx](src/app/page.tsx).
